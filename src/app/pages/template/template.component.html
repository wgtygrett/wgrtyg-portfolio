Example table

Given an array of objects, the table uses the object keys as headers, then displays a row of values for each object.

You can optionally make the column headings stay at the top of the table while scrolling by passing a value of [stickyHeadings]="true".

Code:

<app-table [values]="someArrayOfObjects" [stickyHeadings]="true"></app-table>

Place the dialog component in your templating and make use of the DialogService to handle opening and closing the dialog.

Each dialog must be given a unique id property.

Code:

<!-- component.ts file -->
dialogID: string = 'test-example'

constructor(private dialogService: DialogService) {{ '{' }} {{ '}' }}

openDialog() {{ '{' }}
  this.dialogService.open(this.dialogID)
{{ '}' }}

<!-- component.html file -->
<app-button (click)="openDialog()">Open Dialog</app-button>
<app-dialog [id]="dialogID">
  <div class="p-4 bg-blue-10">
    <h3 class="font-heading text-2xl leading-6 font-medium text-black">
      <fa-icon class="text-blue mr-4" [icon]="infoIcon"></fa-icon>Dialog Title
    </h3>
  </div>
  <div class="p-4">
    <p class="my-2">Here is an example of a dialog.</p>
    <p class="my-2">Use this area to inform the user of the consequences of their actions.</p>
  </div>
  <div class="px-4 py-3 sm:px-6 sm:flex sm:justify-end bg-gray-1">
    <app-button variant="secondary" class="ml-4" (click)="closeDialog()">Nevermind</app-button>
    <app-button variant="primary" class="ml-4" (click)="closeDialog()">Do The Thing</app-button>
  </div>
</app-dialog>
